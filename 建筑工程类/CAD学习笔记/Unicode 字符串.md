# Unicode 字符串

## 1.Unicode 字符串和控制代码

| 名称            | 控制代码 | Unicode 字符串 |
| --------------- | -------- | -------------- |
| 度 (`°`)        | `%%d`    | `U+00B0`       |
| 正/负公差 (`±`) | `%%p`    | `U+00B1`       |
| 直径 (`Ø`)      | `%%c`    | `U+2205`       |

## 2. 文字符号和 Unicode 字符串

| 名称     | 符号 | Unicode 字符串 |
| -------- | ---- | -------------- |
| 几乎相等 | `≈`  | `U+2248`       |
| 角度     | `∠`  | `U+2220`       |
| 下标     | `M₂` | `U+2082`       |
| 平方     | `M²` | `U+00B2`       |
| 立方     | `M³` | `U+00B3`       |
